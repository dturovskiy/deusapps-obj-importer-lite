DeusApps Blender OBJ Importer Lite v1.0.2

Developer:
DeusApps

Changes in v1.0.2:
- Uses user-provided toolbar icons from Logo.zip.
- Icons are now packaged as import_v102, smooth_v102, and clean_v102 to avoid old icon cache collisions.
  - Import
  - Smooth Curves
  - Clean Geometry

Purpose:
Import Blender-exported OBJ geometry into SketchUp cleanly and predictably.

Lite features:
- Geometry-only OBJ import
- Materials/textures intentionally ignored
- Blender scale default
- Orientation presets
- Clean architectural import
- Smooth curves import
- Selected geometry cleanup
- Toolbar with command-specific icons

Recommended Blender export:
- File -> Export -> Wavefront OBJ
- Selection Only: ON
- Apply Modifiers: ON
- Write Materials: OFF
