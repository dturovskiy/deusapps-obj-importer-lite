# Contributing

Thanks for your interest in improving DeusApps Blender OBJ Importer Lite.

This repository is source-available freeware, not an open-source project under an OSI license. Please open an issue before submitting large changes.

Useful issue types:

- OBJ import failure
- Wrong orientation / axis mapping
- Geometry cleanup problem
- SketchUp version compatibility issue
- UI/UX improvement
- Documentation improvement

When reporting an import bug, include:

- SketchUp version
- Operating system
- Blender export settings
- Import preset used
- Screenshot of the result
- A minimal OBJ file if possible
